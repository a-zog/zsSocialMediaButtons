zsSocialMediaButtons zssocial.js

A CSS3 & lightweight JQuery component to quickly display your social profiles.

Social Media Colors

These are default colors for each social media.

You can use them by adding .color-SOCIALNAME class into a text element.

You can also apply .bg-SOCIALNAME (or .border-SOCIALNAME) classes into a container to change the background (or the border) according to the social media's color.

Don't worry about the image icons of the Grid Switcher, they are already base64 encoded inside the JavaScript file.

Advanced use

Advanced configuration is avaible by adding additional data attributes to change Grid Switcher's defaults.

  • Params
  • Example
  • Code
Data attribute Type Default Description
gs-display Integer "grid" Change this attribute to modify the default view mode.
gs-position String "right" Position of the view mode options (right or left).
gs-imgonly Boolean false Change this attribute to show/hide text labels in the view mode options.
gs-zicon Boolean false Activate Zicon font library mashup. It will actually use font-icons instead of those PNG images.
gs-list-icon String zicon-th-list-4 Zicon font icon class of list view.
gs-grid-icon String zicon-th-large-2 Zicon font icon class of grid view.
